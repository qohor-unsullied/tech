Common Docker

September 27, 2022

常見docker指令

Image

list images

docker images

pull images

docker pull debian

pull an image by digest

docker pull ubuntu:20.04

pull from a different registry

docker pull myregistry.local:5000/testing/test-image

remove images

docker rmi [IMAGE_ID]

Continer

run container at backgroud

docker run -d --restart=always --network=host debian

list container

docker ps

execute an interactive shell

# --interactive , -i    Keep STDIN open even if not 
# --tty , -t		    Allocate a pseudo-TTY
docker exec -it [CONTAINER_ID] [bash/sh]

Stop a container

docker stop [CONTAINER_ID]
# --time , -t           Seconds to wait for stop before killing it, default=10
docker stop -t 30 [CONTAINER_ID]
