常見curl指令

POST

application/json

curl -X POST http://localhost:3000/data \
    -d '{"key1":"value1", "key2":"value2"}' \
    -H "Content-Type: application/json"

form data

curl -X POST http://localhost:3000/data \
   -H "Content-Type: application/x-www-form-urlencoded" \
   -d "key1=value1&key2=value2"

with a data fail

curl -d "@data.json" -X POST http://localhost:3000/data
